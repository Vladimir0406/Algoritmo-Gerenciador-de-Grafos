# Algoritmo gerenciador de Grafos
 Um repositório para o desenvolvimento de um trabalho sobre Grafos, abordando conceitos essênciais no tema e os algoritmos de DFS, BFS, Prim, Dijikstra e Kruskal.
 
# Descrição 
 Este Algorimo foi implementado com o objetivo melhorar os conhecimentos da Teoria de Grafos, produzindo uma aplicação capaz de ler, exibir e analizar grafos tanto poderados como não ponderados.
 O algoritmo conta com duas classes principais para o seu funcionamento:
 
- Menu 
  - Responsável pela navegação em todo o código.
  - Possui tratamento de erros ultilizando TryCatch.
  - Alta consistência na funcionalidade e sendo capaz de tratar os diversos erros que um usuário poderia cometer.

- OperacoesGrafos
  - Responsável por toda a parte de leitura, exibição e analize dos grafos presentes.
  - Realiza a aplicação de todos os algoritmos do código.
  - Possui menus para exibição dos grafos altamente interativos

#  Tecnologias

  - .Net
